Socket.IO Setup
================

Nodejs.module requires Socket.IO to function.

Two options for installation:

1. Use npm (Node Package Manager) to install Socket.IO, and symlink into this folder.

> npm install socket.io
> ln -s /path/to/socket/io/install/socket.io /path/to/your/nodejs/module/socket_io/socket.io

OR

2. Download from Socket.IO from http://socket.io into this folder.
